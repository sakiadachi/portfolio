Travel Journal app

overview

Enter your travel destination and date, Travel Journal App returns weather forecast of the location on the date you entered. Besides showing you predicted weather and remained days to your trip, you can make ToDo list for your reminder.

Mood board

Design derives from handwriting, notes and journal. Travel is to make a memory with a organizing.

Design theme

Typography: 'Montserrat', sans-serif

The design idea comes from remembering feeling of satisfaction when you look at an organized notebook and notes highlighted by red and blue color pens. Horizontal line devides sections and your entry on the page.

Working with web API

This app utilises three APIs based on the information user submitted (location and date of trip). The request to web APIs sent from server side code, then dynamically updating UI on client side in order to make API key's anonymous for users, by using "dotenv" module on npm.

  • GeoNames API (api docs)

    Getting Longitude and Latitude of submitted location.

  • weatherbit API (website)

    Getting weather forecast based on the travel date and location you submitted on the browser.

  • pixabay API (api docs)

    Getting image of a travel destination.
